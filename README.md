# Phonepe-Dashboard-On-Power-BI

 Project Overview

The PhonePe Transaction Analysis Dashboard is an interactive Power BI dashboard built to analyze digital payment transactions and generate actionable business insights. The dashboard provides a comprehensive overview of transaction performance, payment success rates, customer behavior, and service-wise revenue distribution through visually appealing and interactive reports.

Designed with a modern UI inspired by the PhonePe theme, this dashboard helps stakeholders monitor key business metrics and make data-driven decisions efficiently.

Objective:-

The objective of this project is to transform raw transaction data into meaningful business insights through an interactive Power BI dashboard. It enables users to monitor payment performance, analyze customer behavior, evaluate service performance, and support strategic business decisions using real-time visual analytics.
